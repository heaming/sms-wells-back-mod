## Requirements

- [JDK 17](https://www.azul.com/downloads/?package=jdk)
- [Maven 3](https://maven.apache.org)

## Install & Execute

### Install

```shell
$ git clone repository-url
```


### Build and test
```shell
$ ./mvnw clean install
```
